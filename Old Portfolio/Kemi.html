Data Science and Frontend projects

Car insurance price index





Once it was decided how this page would look in Figma, I built specific elements, like the calculator and the tool-tip cars, in Visual Studio Code using Javascript, and transferred my code into a shortcode partial within Wordpress, that allowed us to host them on the page. Once the data research was complete, I wrote the copy for the page, along with a press release, and built other visualisations to display key findings, such as the splits by gender.


Scraping the ratings of over 16,000 titles on IMDb and Rotten Tomatoes





I scraped both Rotten Tomatoes and IMDb using the Selenium and Beautiful Soup packages in Python and transferred the resulting CSV files to Google sheets to normalise and get the combined averages. I used this data to write this page and worked with our designer to decorate the page with illustrated elements and visualisations relative to each streaming platform.


Millionaire calculator



I built this millionaire calculator using Javascript, HTML and CSS in VS Code. The backend calculation, based on compound interest, was initially worked out in google sheets and then translated to a formula in Javascript. To build this calculator, I worked closely with our designer to visualise how it would look in Figma.


Decision tree algorithms in R Studio



These are some tree visualisations learned by a decision tree algorithm, which was trained on some data containing variables like humidity, light and more, with the aim of predicting how these all affect the rates of room occupancy. I chose to use the rpart and rpart.plot packages in R for this model as it looked the most aesthetically pleasing, and displayed the data results as clearly and coherently as possible.


London borough pollution visualisation in Tableau



This is an interactive data visualisation based on a dataset regarding pollution levels from 2013 and 2016 in London. It allows users to see how levels developed in their particular boroughs, and how schools in their areas were affected.


Linear and logistic regression models in R Studio



The bottom left linear regression was performed on a dataset in R, and it shows the strong negative relationship between the horsepower and miles per gallon variables. I also plotted the confidence and prediction intervals on the model. The three ROC curve visualisations were derived from a logistic regression model trained on a room occupancy dataset.
